# 何風に作るかを考える - 参考を探す

* 初めから、ゼロから作ろうとするのは難しい。そこで『こんなふうに作りたいなぁ』、『こんな雰囲気が良いな』と思える既存の映像を探す。

* 結婚式で流れるプロフィールムービーなどでは、テレビ番組風にしているものをよく見る。 そんなふうに、自分が目指す映像に近いものはどんなものがあるのか調べる。
